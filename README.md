# formValidatin with vue
its a very simple form which validates with it's own validation's functions

## Getting Started

### requirements
you should have nodejs on your system

### Installing
```
npm install
```
## Deployment localhost:8080 with hot Reload
```
npm run watch
```

## Production Mode
```
npm run start
```
